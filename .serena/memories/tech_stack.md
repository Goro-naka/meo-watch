# MEO Watch - 技術スタック

## フロントエンド
- **Next.js 15**: React フレームワーク（App Routerを使用）
- **React 19**: UIライブラリ
- **TypeScript 5**: 型安全性
- **Tailwind CSS 4**: スタイリング
- **Framer Motion**: アニメーション
- **Lucide React**: アイコン

## バックエンド
- **Next.js API Routes**: バックエンドAPI
- **Supabase**: データベース（PostgreSQL）とAuth
- **Stripe**: 決済処理

## 開発ツール
- **ESLint**: コード品質管理
- **PostCSS**: CSS処理
- **Turbopack**: 高速ビルド

## デプロイメント
- **Vercel**: ホスティングプラットフォーム

## パッケージ管理
- **npm**: パッケージ管理（npm lockを使用）
